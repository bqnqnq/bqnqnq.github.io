# mdBook

### リンク

[mdBook Documentation](https://rust-lang.github.io/mdBook/index.html)

### インストール

Cargo でインストール

```shell
$ cargo install mdbook
```

### 初期設定

`mdbook init` で必要なファイルを配置

```shell
mdbook init [option]
```

[option] を付ける場合、指定ディレクトリで実行

```shell
$ mdbook init bqnqnqbook  # 例
```

`mkbook serve` でローカルサーバーを起動

```shell
# mdbook serve [option]
$ mdbook serve --open  # 例
```
